# 面试小达人 (InterviewKids) - UI 设计规范

## 设计主题
- **应用名称**: 面试小达人
- **目标用户**: 3-8岁儿童家长
- **核心功能**: 儿童教育面试准备、AI模拟面试、成长记录

## 设计风格
- **整体风格**: 现代化Gen-Z爆款风格，活力十足
- **主色调**: 渐变紫 (#7C3AED) → 青色 (#3B82F6) → 蓝绿 (#06B6D4)
- **字体**: Plus Jakarta Sans / Lexend (现代圆润)
- **圆角**: 圆角设计 (16px, 全圆角)
- **效果**: 玻璃态效果(Glassmorphism)、渐变背景、浮动元素

## 页面清单

### 1. 欢迎页 (Welcome)
- 渐变紫-青背景
- 浮动星星、灯泡图标
- "面试小达人"标题
- "立即开始" / "账号登录"按钮
- 玻璃态卡片

### 2. 登录页 (Login)
- 渐变背景
- 玻璃态卡片
- 手机号输入 + 验证码
- 微信一键登录
- 服务条款勾选

### 3. 注册页 (Signup)
- 欢迎文案
- 手机号输入
- 验证码输入
- 下一步按钮
- 服务条款

### 4. 孩子资料-步骤1 (Profile Step 1)
- 6个可爱头像选择
- 孩子姓名输入
- 下一步按钮
- 柔和粉彩色

### 5. 孩子资料-步骤2 (Profile Step 2)
- 兴趣选择标签: 阅读、音乐、绘画、运动、科学、数学、英语、编程
- 年级选择: 小班、中班、大班、一年级、二年级
- 完成按钮

### 6. 家长仪表板 (Dashboard)
- 孩子头像、姓名、年级
- 进度环显示完成进度
- 快捷功能卡片:
  - 模拟面试 (橙色)
  - 面试技巧 (绿色)
  - 成长记录 (蓝色)
  - 家长课堂 (紫色)

### 7. 模拟面试页 (Mock Interview)
- 孩子头像
- 麦克风按钮
- "开始模拟面试"按钮
- 最近练习列表
- 计时器显示

### 8. 课程/课时页 (Lessons)
- 视频缩略图
- 课时标题: "自我介绍技巧"
- 进度条
- "开始学习"按钮
- 课时列表侧边栏

### 9. 面试结果页 (Interview Result)
- 总分圆形显示 + "优秀"评级
- 分项卡片:
  - 表达能力 (进度条)
  - 逻辑思维 (进度条)
  - 自信度 (进度条)
  - 应变能力 (进度条)
- AI反馈摘要
- "再次练习" / "查看详情"按钮

### 10. 面试技巧页 (Interview Tips)
- 文章卡片网格布局
- 缩略图、标题、分类标签、阅读量
- 示例:
  - 幼升小面试常见问题
  - 如何培养孩子表达能力
  - 面试穿着建议
  - 缓解孩子紧张情绪

### 11. 练习历史页 (Practice History)
- 练习记录列表
- 每条记录: 日期、题型、分数、时长
- 状态标签: 良好、优秀、需改进

### 12. 家长课堂页 (Parent Coaching)
- 视频课程卡片
- 缩略图、标题、时长、进度指示
- 示例:
  - 与孩子有效沟通
  - 了解幼升小政策
  - 培养自信心
  - 面试准备清单

### 13. 设置页 (Settings)
- 头像、姓名、会员信息
- 菜单项:
  - 个人信息
  - 孩子管理
  - 通知设置
  - 隐私政策
  - 帮助中心
  - 退出登录
